
# 이분 탐색(이진 탐색, Binary Search)
- 만약 해시 탐색으로도 해결되지 않는 입력 범위라면? -> "이분 탐색"
  - **O(logN)**의 시간 복잡도를 가짐
- 단, 데이터가 정렬되어있어야 함 
- 중요한 특징 중 하나 : **lower bound & upper bound**
